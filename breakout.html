         This is the arcade game Breakout built in Java for my AP CS class my sophomore year. The IDE that you see in the video is BlueJ. While we were given basic instructions like how to arrange the bricks and move the ball, a lot of the features are my own. This is actually my first project where I used file writing and reading, and you can see that working with the highscore.
