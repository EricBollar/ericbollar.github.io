         This game is made to fit the popular '.io' genre of web games. I used Three.js for the visuals and an AWS instance with web sockets for the networking. I have shut down the pirateShips.io website as it was costing me a fair amount to maintain. This was definitely one of my more difficult projects, but I learned a lot in creating it.
